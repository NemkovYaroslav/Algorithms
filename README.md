Описание методов:
1.  Array() - конструктор по-умолчанию, создает массив с текущим числом элементов 0 и запасом 8.
2.  Array(int capacity) - конструктор с параметрами, создает массив с текущим числом элементов 0 и запасом capacity.
3.  Array(const Array& other) - копирующий конструктор, выделяет новую область памяти размерности копируемого массива other.capacity_ и через механизм размещающего new(...) перепысывает в новую память элементы копируемого массива
4.  Array& operator = (const Array& other) + void swap(Array& other) - копирующий оператор присваивания, реализован через идиому Copy-Swap Idiom. Внутри оператора приваивания Array& operator = (const Array& other) создается локальная копия копируемого объекта, через вызов функции swap(...) происходит смена данных текущего и локального объектов, далее из-за выхода из зоны видимости локальная копия удаляется.
5.  Array(Array&& other) - перемещающий конструтор, позволяет захватить R_Value значение. Внутри конструктора происходит перенастройка указателя и "затирание" старого указателя (указание на nullpoiter).
6.  Array& operator=(Array&& other) - перемещающий оператор присваивания, то же самое что и при перемещающем конструкторе, только вызывает вначале освобождение старой памяти массива.
7.  Array operator+(const Array& other) - оператор сложения двух массивов, работает как вставка сначала элементов первого массива, затем в продолжение вставка элементов второго массива.
8.  ~Array() - деструктор, разбирает объекты в ячейках массива, а затем освобождает аллоцированную память
9.  const T& operator[] (int index) const, T& operator[] (int index) - операторы индексирования
10. int insert(int index, const T& element) - вставка некоторого элемента по индексу в массив c сдвигом старых элементов вправо, если запаса памяти хватает, то через механизм размещающего new(...) размещает объект в ячейке массива по заданному индексу, если памяти недостаточно, то аллоцирует новый блок памяти, расширяя запас памяти в 2 раза
11. int insert(const T& element) - вставка некоторого элемента в конец массива
11. void remove(int index) - удаление элемента массива по индексу, оставшиеся элементы сдвигаются влево
12. class BaseIterator - класс базового итератора, содержащего все основные методы итераторов, такие как:
  12.1  Конструктор BaseIterator(...)
  12.2  Взять текущий указатель get(...)
  12.3  Сдвиг к следующему элементу next(...)
  12.4  Проверка наличия следующего элемента hasNext(...)
  12.5  Содержит переопределяемый метод virtual void set(...) = 0; для своих наследников
13. class Iterator final : public BaseIterator - класс стандартного итератора, переопределяет метод set(...) и делает доступным его использование
14. class ConstIterator final : public BaseIterator - класс константного итератора, переопределяет метод set(...) и запрещает его использование
Модульные тесты сделаны на базе CppUnitTestFramework и содержат проверку всех методов
